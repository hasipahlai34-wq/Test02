# 示例文档目录

此目录存放用于演示的示例文档:

## 推荐准备

1. **finance_report.pdf** — 一份包含结构化财务数据的 PDF 报告
   - 包含: 营收数据、增长率、业务线分析
   - 适合演示: 复杂数据查询、多步推理

2. **tech_blog.md** — 一篇技术博客文章
   - 包含: 架构设计、技术选型、代码示例
   - 适合演示: 技术问答、概念对比

## 获取示例文档

你可以使用任何 PDF/Word/Markdown 文档，或从以下来源获取:
- 公司年报 (上市公司 IR 页面)
- 技术白皮书
- GitHub README 文件

启动后，在 Streamlit UI 的"文档管理"页面拖拽上传即可。
